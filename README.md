# Swing_Log

## サイト概要
### サイトテーマ
ゴルフのプレー記録や練習内容を共有し、ユーザー同士で情報交換やモチベーション向上ができるゴルフ特化型SNS
​
### テーマを選んだ理由
私はゴルフをする中で、自分のスコアや練習内容を振り返りたいと思うことが多くありました。
しかし、既存サービスでは「スコア管理」「練習記録」「ユーザー同士の交流」が別々になっていることが多く、まとめて管理しづらいと感じていました。
また、他の人がどのようなゴルフ場へ行っているのか、どのような練習をしているのかを知ることで、自分のモチベーション向上や新しいゴルフ場を知るきっかけになると考えました。
そこで、ゴルフ場の風景やスコア、練習内容を気軽に記録・共有でき、ゴルフ好き同士が交流できるサービスがあれば便利だと考え、このテーマにしました。
​
### ターゲットユーザ
・ゴルフのスコアや練習内容を記録したい人
・ゴルフ仲間と情報共有をしたい人
・他の人の練習方法やゴルフ場の情報を参考にしたい人
・ゴルフのモチベーションを維持したい人
・打ちっぱなしでの練習記録を残したい人
​
### 主な利用シーン
・ゴルフ場でプレーした記録を残したい時
・スコアやラウンド結果を振り返りたい時
・打ちっぱなしでの練習内容を記録したい時
・他ユーザーのおすすめゴルフ場を知りたい時
・ゴルフ仲間とプレー内容や感想を共有したい時
